#python_oscilloscope

#Introduction

Projet python réalisé avec la librairie Tkinter ayant pour but de
recréér un oscilloscope pouvant visualiser la courbe de lissajoux de
deux signaux.

#Prérequis
Avoir python 2.x et Tkinter installé

#Que peut-on faire avec
* Changer la fréquence, l'amplitude et la phase d'une courbe
* Afficher la courbe de lissajoux
* Charger/Sauvegarder un graphe

Ne vous fiez pas à cet outil, il à été créé afin d'apprendre à
utiliser Tkinter
